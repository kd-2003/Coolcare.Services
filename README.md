# CoolCare Services

A full-stack AC Repair & Home Services Booking Platform built with React, Tailwind CSS, Express, MongoDB, and JWT authentication.

## Project Structure

- `/client` — frontend application built with Vite + React
- `/server` — backend API built with Express and MongoDB

## Getting Started

### Backend

1. `cd server`
2. `npm install`
3. Create a `.env` file from `.env.example`
4. `npm run dev`

### Frontend

1. `cd client`
2. `npm install`
3. `npm run dev`

## Features

- User registration and login
- Booking form and service pages
- Admin dashboard for managing services, bookings, customers and reviews
- Technician dashboard with new booking overview
- Responsive UI with Tailwind CSS
- Protected routes and JWT authentication
